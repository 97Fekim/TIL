## 파이썬 대표 라이브러리
- 수치연산 라이브러리 : numpy
```
# list끼리 더한 경우
import numpy as np
list1 = [1,2,3,4]
list2 = [2,4,6,8]

list1 + list2
---------------------------------
[1,2,3,4,2,4,6,8]
```
```
# numpy 배열을 이용해 더한 경우
np.array(list1) + np.array(list2)
---------------------------------
array([3,6,9,12])
```
- 데이터 처리 라이브러리 : pandas
- 차트 그리기 라이브러리 : matplotlib
