# 1. ArrayList
- add(object)로 추가한다.
- get(index)으로 가져온다.
- subList(first, last)를 이용해, 슬라이싱이 가능하다. (String-substring 과 마찬가지.)
- contains(value)를 이용해 리스트가 value를 포함하고 있는지 확인 가능하다.
- remove(index)를 이용해 리스트의 index번째의 value를 삭제한다.
