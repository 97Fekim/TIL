# 단축키
## 이클립스 IDE 단축키
- ctrl + space : 자동완성
- ctrl + F11 : 컴파일 + 실행
- ctrl + / : 주석처리/주석풀기 (//)
- ctrl + shift + / : 주석처리/주석풀기 (/* */)
- sysout + ctrl + space : System.out.println();

## 인텔리제이 IDE 단축키
- Alt + F12 : 터미널 열기
- Alt + 방향키 : 작업창 넘기기
- ctrl + / : 주석처리/주석풀기 (//)
- ctrl + shift + / : 주석처리/주석풀기 (/* */)
