# IDE 단축키
## 이클립스 IDE 단축키
- ctrl + space : 자동완성
- ctrl + F11 : 컴파일 + 실행
- ctrl + / : 주석처리/주석풀기 (//)
- ctrl + shift + / : 주석처리/주석풀기 (/* */)
- sysout + ctrl + space : System.out.println();

## 인텔리제이 IDE 단축키
- 터미널 열기 : Alt + F12
- 작업창 넘기기 : Alt + 방향키
- 주석처리/주석풀기 (//) : ctrl + / 
- 주석처리/주석풀기 (/* */) : ctrl + shift + /
- 설정창 열기 : ctrl + alt + s
- 프로젝트 Run : shift + F10
- 프로젝트 Stop : ctrl + F2
- 만능 자동완성 : ctrl + shift + Enter
- Getter, Setter 생성 : alt + insert
